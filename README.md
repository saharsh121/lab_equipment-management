# lab_equipment-management
A website that manage lab equipment



Lab-Equipment-Management/
│
├── public/                 # Static files (CSS, images, JS)
│   ├── css files
│   └── images/
│
├── views/                  # EJS Templates
│   ├── login_student.ejs
│   ├── login_admin.ejs
│   ├── main_student.ejs
│   ├── main_admin.ejs
│   ├── labs/
│   │   ├── physics.ejs
│   │   ├── chemistry.ejs
│   │   ├── biology.ejs
│   │   └── electronics.ejs
│   └── feedback.ejs
│
├── server.js               # Main server file (Express app)
├── package.json            # Node dependencies
└── README.md







Technologies Used
Technology	Purpose
Node.js	Backend runtime
Express.js	Web framework
EJS	Templating engine
MySQL	Database
HTML, CSS, JS	Frontend
Bootstrap / Custom CSS	UI styling





Make sure you have Node.js and npm installed.
npm install


add the database folder in your sql data.
 run the folder in terminal.

 node server.js


 open the browser and run localhost:3000
